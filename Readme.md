## Foodx - Simple Food App with 35k+ Food Data made with React and API based on Spring Boot



### How to Install ?

Go to the API , ETL and ReactApp folder and read Readme.md 

### Features :

- Login /Logout
- Sign Up
- Show foods based on your IP location (near by)
- AutoComplete
- Add Review 
- Show Reviews
